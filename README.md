# starframe-vault - Star Frame VaultProgram

A vault program built with the Star Frame framework for Solana.

## Getting Started

### Prerequisites

- Rust 1.84.1+
- Solana CLI tools
- Star Frame CLI

### Building

```bash
starpin build
```

### Testing

```bash
starpin test
```

### Deploying

To devnet:
```bash
starpin deploy
```

To mainnet:
```bash
starpin deploy --mainnet
```

### Generate IDL

```bash
starpin idl
```
